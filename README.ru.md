# component-forge

🌐 [English](README.md) | Русский

> CLI инструмент для масштабируемых React проектов с акцентом на архитектуру.

Инструмент, который обеспечивает структурную дисциплину через готовые архитектурные шаблоны — Feature-Sliced Design (FSD) и модульную архитектуру.

---

## Зачем это нужно

React даёт гибкость — но не структуру.

В растущих командах это часто приводит к:

- непоследовательным структурам папок
- размытым границам между слоями
- тесной связанности между фичами
- архитектурной деградации со временем

**component-forge** предоставляет:

- стандартизированную структуру проекта
- чёткие архитектурные слои
- публичные API через index файлы
- предсказуемую стратегию масштабирования

Это не просто инструмент для генерации файлов. Это **инструмент контроля архитектуры**.

---

## Целевая аудитория

- React команды (3–10 разработчиков)
- Растущие стартапы
- Разработчики, которые уже сталкивались с архитектурным хаосом
- Инженеры, которые ценят структуру и долгосрочную поддерживаемость

> Не предназначен для новичков, изучающих основы React.

---

## Поддерживаемые архитектуры

### 1️⃣ Feature-Sliced Design (FSD)

Генерирует слоистую структуру:

```
src/
 ├─ app/
 ├─ processes/
 ├─ pages/
 ├─ widgets/
 ├─ features/
 ├─ entities/
 └─ shared/
```

Обеспечивает:

- изоляцию слоёв
- публичный API через index файлы
- инкапсуляцию фич
- предсказуемые правила масштабирования

### 2️⃣ Модульная архитектура

Генерирует доменно-ориентированную структуру:

```
src/
 ├─ modules/
 │   ├─ auth/
 │   ├─ profile/
 │   └─ dashboard/
 ├─ shared/
 └─ core/
```

Фокус на:

- разделении по доменам
- границах внутри модулей
- масштабируемом владении кодом по командам

## ⚙️ Установка

```bash
# через npm
npm install -g component-forge

# через npx (без установки)
npx component-forge init --architecture fsd
```

---

## Команды

### Инициализация структуры проекта

```bash
component-forge init --architecture fsd
component-forge init --architecture modular
```

### Генерация слайса / модуля

```bash
component-forge generate feature auth
component-forge generate entity user
component-forge generate module profile
```

### Генерация компонента

```bash
component-forge generate component Button
component-forge generate component forms/Input
```
